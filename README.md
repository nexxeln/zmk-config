# Corne v4 ZMK Config

ZMK firmware for Corne v4 with nice!nano v2 and nice!view displays.

## Features

- CAGS home row mods (Cmd on middle finger for Mac)
- Urob's timeless timing (280/175/150ms)
- Hyper key on right inner thumb
- HJKL navigation on Raise layer
- Conditional Adjust layer (Lower + Raise)

## Layers

See `config/corne.keymap` for full layout.

## Build

Push to trigger GitHub Actions build, download firmware from Actions tab.

## Flash

1. Connect half via USB
2. Double-tap reset (appears as USB drive)
3. Copy `.uf2` to drive
4. Repeat for other half
